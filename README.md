# 🐾 Shelter — Cozy House

A responsive two-page website built with semantic HTML, SCSS, BEM methodology, and Vanilla JavaScript. Developed as part of the RS School Front-End course with a focus on clean architecture, responsive design, and reusable UI components.

## ✨ Features

### Responsive Layout

- Fully responsive interface optimized for screen sizes from **320px** to desktop.
- Layout implemented according to the provided Figma design.
- Semantic HTML5 markup for improved accessibility and SEO.
- Modular SCSS architecture using variables, mixins, and the **BEM** methodology.

### Interactive Functionality

- **Responsive burger menu** for mobile navigation.
- **Infinite carousel slider** with smooth transitions.
- **Dynamic pagination** for generating and navigating pet cards.
- **Custom modal component** implemented from scratch for displaying pet information.
- Pet data stored in a **Map** to provide constant-time (`O(1)`) lookup when opening modal windows.

### JavaScript

- Built entirely with **Vanilla JavaScript** without external UI libraries.
- Modular code structure with reusable components and separation of concerns.
- DOM manipulation and event handling implemented from scratch.
